Install with Homebrew

One command. No "app is damaged" warning. 

brew install --cask blackmagic-ai/tap/blackmagic-ai

Direct download

macOS says "BlackMagic AI is damaged"?

macOS Gatekeeper blocks apps downloaded from browsers that aren't signed with an Apple Developer ID. Either install with Homebrew above (recommended — Homebrew handles this automatically), or run this after dragging to Applications: 

xattr -cr "/Applications/BlackMagic AI.app"
